# Majestic Boost

Majestic Boost — компактное Windows-приложение для подготовки системы к игре в Majestic / GTA V. Оно показывает понятное подтверждение перед первым применением настроек, запускает оптимизацию с правами администратора и просит перезагрузить компьютер, когда это действительно нужно.

## Что делает приложение

- применяет набор системных и игровых настроек из `MaxFPS-Apply.ps1`;
- сохраняет исходные значения перед изменением и поддерживает транзакционное восстановление при отмене или ошибке;
- не повторяет уже применённые изменения без необходимости;
- позволяет оставить запущенными Discord, Epic Games Launcher и Steam;
- запускает игровой boost из `Game-Boost.ps1`;
- при каждом запуске проверяет подписанный манифест обновления;
- показывает реальный процент скачивания и установки обновления;
- устанавливает обязательное обновление только после проверки RSA-подписи манифеста, версии, размера и SHA-256 установщика;
- после успешного обновления предлагает одной кнопкой продолжить работу в новой версии.

Приложение не является частью Majestic и не подменяет файлы игры. Некоторые операции меняют системные параметры Windows и требуют подтверждения UAC.

## Сборка

Требуется Windows с .NET Framework 4.8 и 64-битным Framework C# compiler.

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\build.ps1
```

Результат появится в `dist`:

- `MajesticBoost.exe` — переносимая версия;
- `MajesticBoost-Setup-1.4.1.exe` — установщик.

## Выпуск обновления

1. Соберите и полностью проверьте новую версию.
2. Обновите версию в приложении, установщике и manifest-файлах.
3. Рассчитайте SHA-256 и размер нового установщика и обновите `update.json`.
4. Подпишите точные байты `update.json` скриптом `tools/Sign-UpdateManifest.ps1`.
5. Опубликуйте установщик, `update.json` и `update.json.sig` одним проверенным изменением.

Приватный ключ подписи хранится только локально в зашифрованном DPAPI-файле и никогда не должен добавляться в репозиторий. Приложение содержит только публичный ключ.
